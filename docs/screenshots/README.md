# Screenshot Capture Guide

Store release screenshots for Hackloi AI Cyber Lab in this folder.

Recommended filenames:

- `dashboard-overview.png`
- `chat-panel.png`
- `code-workspace.png`
- `scan-analyzer.png`
- `agents-page.png`
- `model-manager.png`

## Clean Capture Setup

Before capturing screenshots:

1. Launch the app with local Ollama running.
2. Use the files in [`../../demo-data/`](../../demo-data/) so the UI looks populated and consistent.
3. Keep the window close to its default desktop size.
4. Use a clean session with readable content and no error toasts visible.
5. Make sure the top bar still shows `Hackloi AI Cyber Lab` and the visible version.

## Suggested Capture Order

1. `dashboard-overview.png`
   Show telemetry, runtime models, workspace summary, and recent jobs.
2. `chat-panel.png`
   Show a useful assistant response with code formatting and the agent router visible.
3. `code-workspace.png`
   Show the file tree, Monaco editor, and code assistant output.
4. `scan-analyzer.png`
   Show structured findings after parsing the demo scan text.
5. `agents-page.png`
   Show the built-in agents, statuses, and per-agent model assignment.
6. `model-manager.png`
   Show installed models, recommended models, and the setup wizard.

## Presentation Notes

- Prefer full-window screenshots over cropped fragments.
- Keep the cyber-lab theme intact.
- Use realistic but safe sample data only.
- Avoid capturing any personal file paths or unrelated terminal history.
