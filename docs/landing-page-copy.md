## Headline

Your Local AI Cybersecurity Workstation

## Subheadline

Hackloi AI Cyber Lab combines local AI chat, code assistance, scan analysis, and tool workflows in one private desktop app powered by Ollama.

## Short Description

Hackloi AI Cyber Lab is a local-first desktop workstation for cybersecurity learning, technical analysis, and development workflows. It keeps models local, keeps files local, and gives users one place to chat, inspect code, review scans, and organize findings.

## Key Features

- Local AI chat with coding, analysis, and documentation workflows
- Code Workspace with Monaco Editor and lightweight IDE behavior
- Scan Analyzer for structured review of common recon and web scan outputs
- Multi-agent support with Coding, Analysis, Documentation, and Coordinator roles
- Explicit tool execution for `nmap`, `ffuf`, `httpx`, `subfinder`, `nikto`, `whois`, and `dig`
- Dashboard telemetry for runtime, models, system state, and recent jobs

## Privacy Copy

Hackloi AI Cyber Lab is built for local-private workflows:

- models run through local Ollama
- no cloud account is required
- no remote file upload is performed
- tool execution stays explicit and user-triggered
