# Release Description

## Title

Hackloi AI Cyber Lab 0.1.0

## Short Description

A local-first AI cybersecurity workstation for coding assistance, scan analysis, tool workflows, and multi-agent review powered by Ollama.

## Key Capabilities

- local AI chat with streaming responses and code formatting
- lightweight code workspace with Monaco Editor
- structured scan analysis for common recon and web outputs
- specialist local agents for coding, analysis, documentation, and coordination
- explicit cybersecurity tool execution with visible command previews

## Why It Is Useful

Hackloi AI Cyber Lab gives users one desktop environment for local development, technical analysis, and note generation. It reduces context switching while keeping model selection, file handling, and command execution visible and user-controlled.

## Privacy Statement

Hackloi AI Cyber Lab is designed for local-private use:

- Ollama runs on loopback
- user files stay local
- no cloud workflow is required
- no hidden execution is performed
